D2 Grup Kurumsal Web Sitesi

Site, frontend/ klasöründeki modern React (Vite + Tailwind) uygulaması olarak geliştirilmektedir. Aşağıdaki adımlarla çalıştırabilirsiniz:

1. cd frontend
2. npm install
3. npm run dev — geliştirme sunucusu (http://localhost:3000)
4. npm run build — üretim derlemesi (frontend/dist)
